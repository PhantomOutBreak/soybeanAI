How it works

In real production, a conveyor + camera + CNN/Transformer model is used to detect defects (fungus, cracks, abnormal color), then sort using pneumatic solenoids/robotic arms.

1) Take a photo

Even lighting, contrasting background (e.g. black/white), reduce shadows and reflections.

2) Analyze with AI

Classify pass/fail, defect locations, and confidence scores.

3) Automatic sorting

Connect PLC/robot to eject failed seeds from the conveyor.

Frequently Asked Questions

Does this demo use real AI?

This is a frontend simulation for UI/UX demonstration only. For real systems, connect to a model API (e.g. ONNX/TensorRT/TF Serving) via HTTPS.

How is it suitable for factory use?

UI is optimized for high contrast, large buttons, and includes offline mode + PWA to reduce network dependency.

What about image privacy?

This demo processes everything in the browser, does not send files to the server, and you can click “Clear All” to remove data from this page instantly.
